# 🌦️ Weather App

A modern and responsive **Weather App** built using **HTML, CSS, and JavaScript** 🌍  

This application provides real-time weather updates including temperature 🌡️ , humidity 💧, and wind speed 🌬️ for any city around the world. It fetches live data from a weather API and displays it in a clean, user-friendly interface.

---

## ✨ Features

- 🔍 Search weather by city name  
- 🌡️ Real-time temperature updates  
- 💧 Shows humidity percentage  
- 🌬️ Displays wind speed  
- 📱 Fully responsive design  
- ⚡ Fast and smooth performance  

---

## 🛠️ Technologies Used

- 🌐 HTML  
- 🎨 CSS  
- ⚡ JavaScript  
- 🔗 Weather API
